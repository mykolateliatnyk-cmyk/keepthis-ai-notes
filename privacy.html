Privacy Policy

Last updated: June 2026

Keep This is a desktop sticky notes app for Mac. We take your privacy seriously. This policy explains what data we collect, how we use it, what is sent to third-party services, and what stays on your device at all times.

Data stored on your device

All your notes are stored locally on your Mac. We do not upload, sync, or transmit your note content to any server — except when you explicitly trigger an AI feature (described below).

The following data is stored locally in the app's data folder (~/Library/Application Support/Keep This/):

  • Note content, font preferences, skin settings, and window positions
  • Daily prediction text (generated once per day and cached locally — not re-sent)
  • App settings: launch at login, dock visibility, language preference
  • An anonymous device identifier stored in the macOS Keychain, used solely for rate limiting AI requests. This identifier contains no personal information and cannot be used to identify you.

AI text editing features Claude API

Keep This includes optional AI-powered text actions: Improve, Shorten, Expand, Motivate, Convert to Tasks, and a free-form custom prompt input. These features are only activated when you explicitly tap an AI action button.

When you use an AI feature, the text content of your current note is transmitted securely to our proxy server and forwarded to Anthropic's Claude API for processing. No other data is sent.

How your input is protected:

  • The app's AI system includes a built-in content filter. Requests that are not related to note text editing are automatically rejected by the AI — the model is instructed to respond only to text editing tasks.
  • The free-form input field is limited to 200 characters and is subject to the same content filter as preset actions.
  • Your note text is not stored on our servers. Requests are processed in real time and immediately discarded after a response is returned.
  • We do not use your note content to train AI models. Anthropic's data handling policies apply to API calls — see anthropic.com/privacy.
  • AI-generated content is returned as a suggestion. You can undo any AI transformation using the "Restore original" button that appears immediately after each action.
  • AI features require an active internet connection. If no connection is available, no data is sent and the feature will not activate.

Daily Prediction feature Claude API

Once per day, Keep This generates a short inspirational prediction using AI. This feature sends only a fixed generic prompt to the Claude API — no personal information, no note content, and no user-identifiable data is included in this request.

The generated prediction is cached locally and displayed for the remainder of the day. No additional requests are made until the next day.

Subscriptions & payments Apple StoreKit

Keep This Pro subscriptions are processed entirely by Apple through the App Store using StoreKit 2. We do not collect, store, or have access to your payment information, credit card details, or billing address.

All subscription management is handled through your Apple ID. You can view, modify, or cancel your subscription at any time in System Settings → Apple ID → Subscriptions.

Data we do not collect

  • Your name, email address, or any account information
  • Analytics, usage statistics, or behavioral data
  • Crash reports
  • Location data
  • Your IP address (our proxy does not log IP addresses)
  • Data from other apps on your Mac
  • Note content outside of explicit AI feature usage

Third-party services

Keep This uses the following third-party services. No personally identifiable information is shared with any of them:

  • Supabase — serves as a secure API proxy between Keep This and the Claude API. Supabase does not store your note content. Learn more at supabase.com/privacy.
  • Anthropic (Claude API) — processes AI text editing requests and generates daily predictions. Learn more at anthropic.com/privacy.
  • Apple App Store — handles all in-app purchases and subscription billing via StoreKit.

Data security

  • All communication with external services uses HTTPS/TLS encryption
  • The anonymous device identifier is stored in the macOS Keychain, which is encrypted and protected by the operating system
  • Note content is never written to any external storage or database
  • We do not have the technical ability to access your locally stored notes

Children's privacy

Keep This is not directed at children under the age of 13. We do not knowingly collect any information from children. The AI features include content filtering that rejects off-topic or harmful prompts. If you believe a child has used the app in a way that raises privacy concerns, please contact us.

Your rights

Since we do not collect personal data, there is no user account or profile to delete. All locally stored data (notes, settings, predictions) can be removed by uninstalling the app and deleting the app's data folder at ~/Library/Application Support/Keep This/.

The anonymous device identifier in the Keychain can be cleared by uninstalling the app.

Changes to this policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated date. Continued use of the app after changes constitutes acceptance of the updated policy. Significant changes will be communicated through the app.
